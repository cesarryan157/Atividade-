# TechManage Backend

## Requisitos
- Node.js >= 18
- MySQL
- npm

## Instalação
1. clone repo
2. cp .env.example .env e configure
3. npm install
4. npx sequelize db:create
5. npm run migrate
6. npm run seed
7. npm run dev
